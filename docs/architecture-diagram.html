NeuroFlow Core Architecture

Telegram user segmentation & AI orchestration toolkit — v0.1.0

Telegram Ingestor

  • • Polls Telegram Bot API via getUpdates
  • • Classifies messages into 16 trigger types
  • • SQLite UserStore for profiles + events
  • • REST API :8888 + HTML dashboard
  • • Configurable poll interval & thresholds

Telegram Segmentation

  • • Deterministic FSM — 7 states + CONVERTED
  • • 16 mapped triggers for state transitions
  • • Automatic decay: cold at 7d, churn at 30d
  • • Priority scoring: HOT=100, WARM=70, …
  • • export_json(), segment_counts(), hot_leads()

OSW Engine

  • • DAG with Kahn's topological sort
  • • GraphMemory — trust-decaying fact store
  • • StateMachine — agent lifecycle (6 states)
  • • AgentCard — registration with retry budget
  • • OSWEngine: goal → decompose → execute → report
